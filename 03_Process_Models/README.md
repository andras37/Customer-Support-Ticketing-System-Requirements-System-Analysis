# Process Models

This folder contains BPMN process models created as part of the
Customer Support Ticketing System analysis.

## Included artifacts
- End-to-end BPMN diagram of the ticket handling process
- Role-based swimlanes (Agent, Manager)
- Decision gateways for information completeness and issue resolution

## Purpose
These models demonstrate:
- Process understanding from a Business Analyst perspective
- Ability to translate operational workflows into BPMN
- Structured, system-ready process documentation
